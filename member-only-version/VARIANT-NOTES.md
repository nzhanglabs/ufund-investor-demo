# Two-Level Variant V2 (Teaser + Sidebar Drilldown)

This folder is a separate copy and does not overwrite previous versions.

## Roles
- Qualified Access
- Club Member

## Key UX updates
- Qualified users can see a `Club Member Access` tab and member module options as preview-only.
- Clicking `Club Member Access` as qualified shows a lock message and teaser modules.
- `Start Here` is role-specific welcome content (qualified/member) with CTA.
- Sidebar removes `Resource Center` and adds grouped subtabs under `Qualified Access` and `Club Member Access`.
- Added `FAQ` and `Contact Us` pages.
- Added `NEW` badges for Newsletter/Reports/Video/Events in sidebar.
- Added event sign-up demo page and button.
- Priority Q&A page includes IR contact details (Jackie / Momo / Lisa).
