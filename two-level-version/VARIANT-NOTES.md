# Two-Level Variant

This folder contains the 2-level access model variant for team review.

## Roles
- Qualified Access (combined original Level 1 + Level 2 access)
- Club Member (original Level 3 access)

## Key file changed
- app.js

## How to run this variant
From project root:
```bash
cd two-level-version
python3 -m http.server 8010
```
Open:
- http://localhost:8010/index.html
